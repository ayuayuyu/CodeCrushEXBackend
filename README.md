# Nitro starter

Look at the [nitro quick start](https://nitro.unjs.io/guide#quick-start) to learn more how to get started.


### 開発環境
**uuid**

```bash
pnpm install uuid
```

```bash
pnpm i nanoid
```

**SQLite**

```bash
pnpm install better-sqlite3
pnpm install --save-dev @types/better-sqlite3
```
```bash
pnpm i -D wrangler
```
**起動**

```bash
pnpm dev
```
